### 问题描述

> https://leetcode-cn.com/problems/largest-perimeter-triangle/

给定由一些正数（代表长度）组成的数组 A，返回由其中三个长度组成的、面积不为零的三角形的最大周长。如果不能形成任何面积不为零的三角形，返回 0。

```
输入：[2,1,2]
输出：5

输入：[1,2,1]
输出：0

输入：[3,2,3,4]
输出：10

输入：[3,6,2,3]
输出：8
```

提示：

```
3 <= A.length <= 10000
1 <= A[i] <= 10^6
```

### 思路

* 三角形组成条件：短边之和大于第三边

* 题目转变成：找出和最大的三个数。并且三个数满足，两个短边之和大于第三边
