### 问题描述

> https://leetcode-cn.com/problems/same-tree/

给定两个二叉树，编写一个函数来检验它们是否相同。如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

示例:

```
输入:    1         1
        / \       / \
       2   3     2   3

       [1,2,3],   [1,2,3]

输出: true
```

### 思路

树的遍历

* 前序遍历：根结点 -> 左子树 -> 右子树

* 中序遍历：左子树 -> 根结点 -> 右子树

* 后序遍历：左子树 -> 右子树 -> 根结点

* 层次遍历：只需按层次遍历即可
